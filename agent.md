Yes, this is an agent, specifically a routing agent (sometimes called a tool‑using router or orchestration agent)

It’s an agent because it:

- Interprets the user query,
- Chooses a tool/route,
- Executes and returns.
That’s agentic, but minimal—there’s no iterative planning or self‑reflection.
